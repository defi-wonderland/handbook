# Knowledge & Docs

These tools help us share information, document our work, and stay coordinated across teams. Most of what you’ll need is already written down — and if it’s not, you can help make it clearer for everyone else.

## Notion: the knowledge hub

We use **Notion** to document how things work at Wonderland. This includes:

- Internal processes and workflows
- Project-specific documentation
- Onboarding material
- Partner-facing pages when needed

Use Notion as your first stop before asking operational questions — it might already be covered.

If you’re managing a process, building new infra, or creating shared templates, update Notion so the next person doesn’t have to guess.

## Google Drive: shared files

**Google Drive** is where we keep shared documents, partner materials, and working drafts. You’ll use it to access:

- Project docs and specs
- Templates and assets
- Partner-facing work or delivery files

:::tip SECURITY
Everytime you want to open a PDF, we recommend you to upload it to Drive and then open it. Stay safe : )
:::

Before sharing a file, check the permissions:
- Avoid public links unless strictly needed
- Limit editing rights when possible

## Google Calendar: scheduling

We use **Google Calendar** to stay aligned on meetings, availability, and events.

When you join:

- Set your **working hours** and any block of time where you are away from keyboard.
- Accept invitations to the shared calendars:
    - `🗓 General` — announcements and company-wide events  
        - `👥 Team Meetings` — biweekly syncs (recommended)  
        - `🎮 Game Day` — optional social events  
        - `🎙 Podcasts` — optional learning sessions (usually recorded)
    - `🚫 OOO` — synced with Float

Don’t delete shared events, even if you’re not attending.

You’ll also have regular **1-1s with the COO** every three weeks. If you haven’t received an invite, just ask.

## Google Tasks: for personal tracking

**Google Tasks** is available in Gmail and Calendar. It’s helpful for keeping track of:

- Personal to-dos
- Project checklists
- Quick assignments

You can:
- Assign tasks to others at Wonderland
- Set deadlines and times that show directly in your calendar

Use it if you want a lightweight way to manage small follow-ups or reminders.

# Welcome to the People Section ✨

This section of the handbook explains how we work together, what tools we use, and how we stay coordinated across projects and teams.

We’re a distributed group, and we rely on a shared setup to keep things smooth. This includes how we show up across platforms and how we handle access and security.

## Staying Consistent

We use a number of different tools across projects, but we keep a few things standard across all of them:

- Use the **same anonymous name** in every tool.
- Use the **same profile picture**.
- Enable **two-factor authentication (2FA)** on every account.

This helps us stay recognizable and secure, especially as we collaborate across teams and time zones.

:::info IMPORTANT 💬 
If something's missing (access, setup, or credentials) message the COO and we’ll sort it out.
:::
